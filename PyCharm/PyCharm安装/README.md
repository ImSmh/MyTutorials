# Ubuntu16 安装 PyCharm
到 https://www.jetbrains.com/pycharm/download/#section=linux 下载 PyCharm 包，此处教程使用版本为专业版  
- 进入 PyCharm 包目录进行解压
```shell
tar xfz pycharm-professional-2019.2.1.tar.gz    
```
- 安装 PyCharm
```shell
cd pycharm-2019.2.1/bin/   
./pycharm.sh      
```
