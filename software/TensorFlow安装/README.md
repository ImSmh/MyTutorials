# Ubuntu16 安装 TensorFlow
硬件：联想 R720，显卡 GTX1050  
### 安装 CPU 版本
- 安装 numpy  
```shell
sudo apt-get install python-numpy  
```
- 安装 pip  
```shell
pip install --upgrade tensorflow #python2  
或
pip3 install --upgrade tensorflow #python3  
```
- 安装 TensorFlow CPU 版本  
```shell
pip install --upgrade tensorflow  #python2
或  
pip3 install --upgrade tensorflow #python3  
```
